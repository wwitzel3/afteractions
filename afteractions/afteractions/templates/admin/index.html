Configure Corp or Alliance

Here is how you do this. Grab a mail that has someone from your corp or alliance on it. Kill or loss, it doesn't matter and post it. Now use the box below to search for your corp or alliance and set it. Simple as that. If you change corps or change alliances and run the killboard and are taking your shit with you, just revist this search box.

Image & Portrait Paths

Image path is relative from your public folder. Usually this will be /images this is the same for portraits, this is usually /images/portraits.
The Portrait Full path should be the full path on disk where you want portraits saved to. This needs to be writable by the server as it is going to create multiple directories there. This usually looks something like this, /home/afteractions/deploy/public/images/portraits
